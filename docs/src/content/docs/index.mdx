## Two ways to install

### Local install (Docker)

<Code
  code={`curl -fsSL https://install.deepsql.ai/install.sh | bash`}
  lang="bash"
  frame="terminal"
/>

Brings up the full stack — frontend, backend, Postgres + pgvector, Valkey — on any Linux or macOS host with Docker. UI at `http://localhost:3035`. Works on Amazon Linux 2023, Ubuntu, Debian, Fedora, Alpine, Arch, openSUSE, macOS, on x86_64 and arm64.

### AWS (one-click CloudFormation)

<LinkButton href="https://console.aws.amazon.com/cloudformation/home?region=us-east-1#/stacks/quickcreate?templateURL=https://deepsql-cfn-461084093078-us-east-2.s3.us-east-2.amazonaws.com/deepsql-stack.yaml&stackName=deepsql-selfhost" variant="primary" icon="external">
  Launch Stack in us-east-1
</LinkButton>

Provisions a `t4g.large` (AL2023, arm64) in your existing VPC — the same one as your Aurora or RDS — and runs the installer automatically. No public IP, SSM-managed, encrypted gp3 root, optional support IAM user for the DeepSQL team. Change the region in the URL (`region=us-east-1` → your region) before clicking. [Full walkthrough](/aws/cloudformation/).
